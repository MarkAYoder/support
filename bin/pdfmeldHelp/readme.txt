FyTek PDF Meld - Linux version
http://www.fytek.com

The documentation is the file pdfmeld.pdf.  You'll need Acrobat or Acrobat
Reader in order to view it.

You can get a free copy at:

http://www.adobe.com/products/acrobat/readstep.html

The documentation contains a section on the executable version of the software.
Use that section to reference the options you may use with the Linux version.

Note that the program works with unencrypted PDFs.
The documentation, pdfmeld.pdf, is encrypted so do not use it in testing.
